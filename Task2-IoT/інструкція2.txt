data_sender.ino

Цей комплект дозволить вам зібрати і зберігати дані на SD-карті без використання Wi-Fi. 
Використовуйте модулі і плату, що найкраще відповідають вашим потребам і можливостям проекту.

Щоб реалізувати проект, який включає збір даних за допомогою Arduino або ESP32 
і збереження даних на SD-карті, вам знадобляться наступні компоненти:

1. Плата
    Arduino Uno:
        Основна плата для простих проектів. Підходить для невеликих датчиків і модулів.
    ESP32:
        Потужніша плата з вбудованим Wi-Fi та Bluetooth. Корисна для проектів, де потрібна додаткова обчислювальна потужність або зв'язок через бездротові технології.

2. Датчики
    Аналоговий сенсор температури (наприклад, LM35):
        Вимірює температуру. Значення може бути перетворено в градуси Цельсія.

    Аналоговий сенсор вологості або рівня води (наприклад, сенсор вологості грунту):
        Якщо потрібно вимірювати рівень вологості або води.

    Датчики потоків води:
        Вимірює витрату води. Наприклад, YF-S201.

3. Модуль SD-карти
    SD Card Module:
        Для зчитування та запису даних на SD-карту. Модулі часто використовують SPI інтерфейс.

4. Допоміжні компоненти
    Провідники (Dupont wires):
        Для з'єднання плат з датчиками і модулем SD-карти.
    Резистори (якщо потрібні для датчиків):
        Деякі датчики можуть вимагати використання резисторів для коректної роботи.
    Блок живлення або батарея:
        Для забезпечення електроживленням плати та датчиків.

5. Інструменти для монтажу
    Бредборд (Breadboard):
        Для тимчасового з'єднання компонентів без пайки.
    Паяльник і припій (якщо потрібно паяти компоненти):
        Для створення постійних з'єднань.

6. Карта пам'яті
    SD карта (на приклад 4GB або 8GB):
        Для зберігання даних. Можна використовувати стандартні карти пам'яті.

7. Заміна датчиків (опційно)
Якщо у вас є специфічні вимоги або умови, вам можуть знадобитися інші датчики:
    Датчики тиску: Для вимірювання тиску води.
    Датчики pH: Для вимірювання кислотності води.
    Датчики рівня: Для визначення рівня води в резервуарах.
